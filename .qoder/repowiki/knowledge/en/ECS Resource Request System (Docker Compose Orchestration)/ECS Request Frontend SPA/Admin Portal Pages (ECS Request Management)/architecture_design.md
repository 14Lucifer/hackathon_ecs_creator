Flat collection of sibling React page components mounted under `/admin/*` routes; `AdminLayout.jsx` is the only shared shell — it renders a fixed left sidebar whose links are driven by the `NAV` config array plus an `<Outlet />` content area. Each page owns its own local state (`useState`/`useEffect`) with no cross-page imports or shared store. Layout-level concerns (current user, logout) come from the app-level `useAuth` hook in `../../App`. All pages consume UI primitives exclusively from `../../components/ui` (`Badge`, `ErrorBanner`, `Modal`, `Spinner`, `StatCard`, `EmptyState`, `useToast`) and call backend endpoints solely through the centralized `api` client in `../../services/api`; there is no direct `fetch` usage. Page-specific modal dialogs (e.g. `ApproveModal`, `RejectModal`, `UserModal`) are declared as inner functions within their parent file and receive minimal props (`mode`/`user`/`requestIds`), keeping inter-page coupling at zero.
