The root composes four services defined in `docker-compose.yml`: `postgres:15-alpine` (health-checked via `pg_isready`) is depended on by `backend/`, which runs the FastAPI app; `frontend/` is built once and its `/output` volume is mounted read-only into `nginx:alpine` at `/usr/share/nginx/html`. Nginx is the only externally exposed port (80) and routes all `/api/*` requests to the backend while serving the SPA for every other path. The frontend's dev server (`vite.config.js`) proxies `/api` directly to `localhost:8000`, but production traffic never reaches the backend except through nginx. Both children share secrets exclusively via `.env` loaded by `docker-compose.yml env_file`; there is no direct inter-service networking between frontend and backend.
