Each file is a self-contained React page component mounted under `/admin/*` routes and rendered inside `AdminLayout.jsx`, which provides a fixed left sidebar (`NAV` config drives all links) plus an `<Outlet />` content area; layout state (current user, logout) comes from the app-level `useAuth` hook in `../../App`. All pages share a common UI contract: they import shared primitives from `../../components/ui` (`Badge`, `ErrorBanner`, `Modal`, `Spinner`, `useToast`) and call backend endpoints exclusively through the centralized `api` client in `../../services/api`. There is no cross-page state — each page owns its own local `useState`/`useEffect` data lifecycle, with child modal components defined inline per page (e.g. `ApproveModal`, `RejectModal`, `UserModal`). Dependency direction is strictly inward: pages → `components/ui` + `services/api` + `App.useAuth`; there are no inter-page imports.
